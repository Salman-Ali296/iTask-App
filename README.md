# 📖 Overview

**iTask App** is a task management application built to help users stay organized and productive by managing their daily activities in one place. Users can create, edit, organize, and track tasks while monitoring their progress through an intuitive and responsive interface.

The application follows modern development practices with a scalable architecture, making it suitable for both personal productivity and collaborative task management.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/your-username/iTask-App.git
```

## Navigate to the Project Folder

```bash
cd iTask-App
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file and add:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

## Run the Development Server

Frontend

```bash
npm run dev
```

Backend

```bash
npm run server
```
